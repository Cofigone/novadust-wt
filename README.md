# Novadust for Windows Terminal

> A color theme inspired by the stars

**NOVADUST** is a dark blue color theme designed for high contrast yet maintaining the softness and warmth necessary for endless, comfortable work.<br>
Created by [marta.stl](https://mmartamg.github.io/novadust/)<br>
Ported for Windows Terminal by me.

**Quick Install:**

1. Copy [`windows-terminal/novadust-windows-terminal.json`](./windows-terminal/novadust-windows-terminal.json)
2. Open Windows Terminal Settings (`Ctrl+Shift+,`)
3. Click "Open JSON file"
4. Add the theme to the `"schemes"` array
5. Apply it in Settings → Profiles → Appearance
