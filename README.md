## Windows Terminal Theme

The NOVADUST theme is now available for Windows Terminal! Get the same beautiful color scheme in your terminal.

**Quick Install:**
1. Copy [`windows-terminal/novadust-windows-terminal.json`](./windows-terminal/novadust-windows-terminal.json)
2. Open Windows Terminal Settings (`Ctrl+Shift+,`)
3. Click "Open JSON file"
4. Add the theme to the `"schemes"` array
5. Apply it in Settings → Profiles → Appearance
